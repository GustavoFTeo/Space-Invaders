# Space Invaders

Jogo Space Invaders, desenvolvido como projeto acadêmico, usando a engine Godot 4.

## Objetivo
Atire nos inimigos com sua nave, até que eles explodam.

Mas cuidade, eles também podem te destruir com seus tiros.
